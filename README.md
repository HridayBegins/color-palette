# Color Palette Generator 🎨

An interactive color palette visualizer built entirely in Excel using VBA macros.

## Preview
> A hexagonal color palette rendered inside Excel with dynamic color swatches.

## Features
- 12 color swatches displayed in a hexagonal palette layout
- Built purely with Excel VBA and macro scripting
- No external tools or plugins required — just Excel

## How to Use
1. Download the `.xlsm` file
2. Open in Microsoft Excel
3. Enable macros when prompted
4. The palette renders automatically

## Built With
- Microsoft Excel
- VBA (Visual Basic for Applications)
- Excel Object Model
